# Aplicación del Cálculo Diferencial en la Optimización de Costos

Proyecto desarrollado para el Trabajo de Investigación Formativa (TIF).

## Tecnologías

- HTML
- CSS
- JavaScript

## Funcionalidades

- Modelado de una función de costos.
- Cálculo automático de la derivada.
- Obtención del punto óptimo de producción.
- Cálculo del costo mínimo.
- Interpretación de los resultados.
